# Victor Muthanga - IT Engineer Portfolio

## Welcome! 👋

This is a modern, minimal portfolio website showcasing my skills, experience, and projects as an IT Engineer.

## 🚀 Features

- **Responsive Design**: Fully responsive and works on all devices
- **Modern & Minimal**: Clean, professional design with smooth interactions
- **Smooth Navigation**: Smooth scrolling and intuitive navigation
- **Skills Showcase**: Comprehensive display of IT skills and expertise
- **Experience Timeline**: Professional experience displayed in an engaging timeline format
- **Contact Form**: Easy way to get in touch
- **Accessibility**: Built with semantic HTML and accessibility in mind

## 📚 Sections

1. **Hero Section**: Eye-catching introduction
2. **About**: Bio and professional summary
3. **Skills**: Organized by categories (Web Dev, Backend, Cloud & DevOps, Tools)
4. **Experience**: Timeline of education and professional experience
5. **Contact**: Contact information and contact form

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling and responsive design
- **JavaScript**: Interactivity and form handling

## 📱 Responsive Breakpoints

- Desktop: 1200px and above
- Tablet: 768px to 1199px
- Mobile: Below 768px

## 🎨 Design Details

- **Color Scheme**: Professional and minimal
  - Primary: #2c3e50 (Dark Blue)
  - Secondary: #3498db (Light Blue)
  - Accent: #ecf0f1 (Light Gray)
